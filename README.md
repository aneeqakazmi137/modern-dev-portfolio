# Modern Developer Portfolio (Next.js)

A high-performance personal portfolio developed using the Next.js framework to demonstrate a transition into modern web development and professional project presentation.

## Technical Framework
* **Core Framework**: Built with Next.js using the App Router architecture for optimized routing and performance.
* **Styling**: Utilizes Tailwind CSS for a professional, utility-first design approach with a focus on dark-mode aesthetics.
* **Typography**: Integrated with Geist font optimization to ensure consistent and high-quality text rendering across all devices.
* **Responsive Design**: Engineered to be fully compatible with desktop, tablet, and mobile viewing environments.

## Deployment and Execution

### Local Development
1. Clone the repository to your local environment.
2. Install dependencies: `npm install`
3. Launch the development server: `npm run dev`
4. Access the application at `http://localhost:3000`.

### Production Deployment
This project is configured for seamless deployment on the Vercel Platform, ensuring global availability and high-speed delivery.
